
 # x402 Integration (beta)

 It is possible to call the subnet generation for flux dev images via x402 protocol. 

 The x402 gateway is at `https://x402.dippy-bittensor.studio` and currently we support USDC payments on Base.


## Instructions
1. Clone the repo at https://github.com/donaldknoller/x402ize
2. Install go dependencies via `go mod download`
3. Setup a keystore via the following:
```
  go run ./cmd/keytool \
    --keystore /path/to/keystore \
    --password-file /path/to/password.txt
```
4. Run the following:
```golang
 go run ./cmd/client \
    --keystore /path/to/keystore \
    --password-file /path/to/password/file.txt \
    --url https://x402.dippy-bittensor.studio/generate \
    --method POST \
    --network base \
    --body '{"prompt":"your prompt here","webhook_url":"https://webhook.url/path/to/destination"}'
```

Note that the image results will be provided to the webhook destination with the image url
